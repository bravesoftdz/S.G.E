###################
What is S.G.E
###################

S.G.E is an Enterprise Management System  - Systeme de Gestion des Entreprise 
dans le secteur de formation professionnelle developpé sous  PHP. 

*******************
Release Information
*******************

This repo contains in-development code for future releases. To download the
latest stable release please visit the `S.G.E Downloads
<https://cfpasidiali.org>`_ page.

**************************
Changelog and New Features
**************************

You can find a list of all changes for each release in the `user
guide change log <https://github.com>`_.

*******************
Server Requirements
*******************

PHP version 5.6 or newer is recommended.

It should work on 5.3.7 as well, but we strongly advise you NOT to run
such old versions of PHP, because of potential security and performance
issues, as well as missing features.
